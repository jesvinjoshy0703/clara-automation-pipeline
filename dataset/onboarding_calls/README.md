# Clara Automation Pipeline

This project simulates an AI-powered onboarding automation system for
commercial customers using the Clara Answering Agent.

## Project Overview
The system processes demo call transcripts from businesses and
automatically generates onboarding accounts.

## Features
- Processes demo call transcripts
- Extracts business information
- Identifies emergency service conditions
- Automates account onboarding
- Creates structured account outputs
