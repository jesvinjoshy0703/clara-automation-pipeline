# Clara Automation Pipeline

This project simulates an AI-powered automation pipeline that processes commercial demo call transcripts and generates onboarding accounts for businesses.

## Features

- Processes demo call transcripts
- Extracts business information
- Generates onboarding accounts
- Supports workflow automation using n8n
- Structured automation pipeline in Python
